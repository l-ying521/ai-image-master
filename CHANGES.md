## 🔧 技术改进点

### 环境变量支持
现在支持通过 `.env` 文件配置以下参数：
- `BAIDU_API_KEY`: 百度 AI API Key
- `BAIDU_SECRET_KEY`: 百度 AI Secret Key
- `FLASK_DEBUG`: 是否开启调试模式（True/False）
- `PORT`: 服务端口（默认 5000）
- `FLASK_HOST`: 监听地址（默认 0.0.0.0）

### 日志系统
